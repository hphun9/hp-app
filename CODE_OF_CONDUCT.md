# Code of Conduct

This project is intended to be a friendly, welcoming place for anyone who wants to use or contribute to the Base App Helm Chart.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of level of experience, technical background, or any personal characteristics.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the community and users
- Showing empathy towards others

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Trolling or flaming
- Publishing othersâ€™ private information without permission
- Any behavior that makes others feel unsafe or unwelcome

## Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.  
They may remove or reject comments, commits, code, issues, and other contributions that are not aligned with this document.

## Scope

This Code of Conduct applies within:

- The project repository and issue tracker
- Pull requests and code review
- Any public spaces where the project is being discussed

## Reporting

If you experience or witness behavior that violates this Code of Conduct, you can:

- Open an issue in the repository (if you feel safe to do so), or
- Contact the maintainer privately (for example, through the contact information in the repository or profile)

All reports will be reviewed, and appropriate action will be taken when necessary.

## Enforcement

Project maintainers will:

- Respect confidentiality requests
- Take reasonable steps to protect reporters from retaliation
- Take actions they consider appropriate, which may include warnings, temporary bans, or permanent bans from project participation

---

By participating in this project, you agree to abide by this Code of Conduct.
